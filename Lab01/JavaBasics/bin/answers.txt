1. "You have chosen: No" pops up when you choose Cancel.
2. Use an Object[] options string array and and add as an argument in JOptionPane.showOptionDialog to use as options, as the example below:

        Object[] options = {"I do", "I don't"};
        int option = JOptionPane.showOptionDialog(null, 
                                                  "Do you want to change to the first class ticket?", 
                                                  null, 
                                                  JOptionPane.YES_NO_OPTION, 
                                                  JOptionPane.QUESTION_MESSAGE, 
                                                  null, 
                                                  options, 
                                                  options[0]);
